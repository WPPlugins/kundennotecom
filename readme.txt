== Description ==

**Empfehlungsmarketing als neuer Vertriebskanal:**
**Das ist kundennote.com!**

In unserer schnellen, digitalen Zeit ist es wichtig, in wenigen Sekunden unternehmerisch zu wirken. Nutzen Sie hierfür eine altbewährte und klassische Marketingstrategie: Empfehlungen. Was passiert, wenn Sie einem Freund einen guten Kinofilm empfehlen? Richtig. Er wird diesen Film wohl sehen. Dieses Wissen ist auch unternehmerisch zu nutzen: Zeigen Sie Ihren potenziellen Interessenten und Besuchern, wie Ihre Kunden denken. Überzeugen Sie mit erbrachter Leistung.

kundennote.com ist ein unabhängiges Bewertungsportal für Unternehmen jeder Art. Wir kombinieren das klassische Empfehlungsmanagement mit einem Gütesiegel für Ihre Website. Das heißt für Sie: Ihre Kunden sehen sofort, dass Sie ein Top Dienstleister sind. Profitieren Sie von den Bewertungen Ihrer Kunden und steigern Sie somit nachhaltig Ihren Umsatz.

* Feedback zu Bewertungen
* Suchmaschienfreundliches Profil
* Authentisches Unternehmensprofil
* Bewertungssiegel für Ihre eigene Website
* Bewertungen und Sterne in Google-Suchergebnissen
* nur für Unternehmen geeignet (Registrierung erforderlich)

Kostenlose Registrierung unter [kundennote.com](https://kundennote.com)

[youtube https://www.youtube.com/watch?v=YYxsmKShues]

== Installation ==

1. Laden Sie das Plugin in den Ordner `/wp-content/plugins/kundennotecom` oder installieren Sie es direkt über den Plugin Installer von WordPress.
1. Aktivieren Sie das Plugin über über die 'Plugins' Seite in WordPress
1. Konfigurieren Sie das Plugin über Einstellungen->kundennote.com
1. Fügen Sie das Bewertungssiegel über die verfügbaren Shortcodes [kundennote] oder [kundennote style="1"], [kundennote style="2"], ... oder über das Widget unter Design->Widgets in Ihre Website ein
